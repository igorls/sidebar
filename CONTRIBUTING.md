# Contributing

Thanks for your interest in Sidebar.

This is a small two-person project and is **source-available, not open source**
(see [LICENSE](LICENSE)). To keep our options open — including possibly
open-sourcing or relicensing it later — we are **not accepting external code
contributions right now**. Outside pull requests can't be merged, because once
a third party's copyright is in the tree we can no longer relicense freely.

You're very welcome to:

- **Open an issue** to ask a question, report a bug, or share feedback.
- **Reach out** to the authors on GitHub:
  [@igorls](https://github.com/igorls) · [@domiscd](https://github.com/domiscd).

If that changes, we'll update this file with how to contribute.
